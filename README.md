### 💰 [WalletWise](https://github.com/Samuel-ahumuza/walletwise_api) — Personal Finance Tracker
A full-stack personal finance app built as a teaching project. Every file is heavily commented to explain *why* the code is written the way it is — not just what it does.

- **Backend:** Express.js · MySQL · JWT Auth · REST API · Joi validation · Cron jobs
- **Frontend:** [`walletwise_ui`](https://github.com/Samuel-ahumuza/walletwise_ui) — JavaScript · HTML · CSS
